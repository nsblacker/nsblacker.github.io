Tesla K40c加速卡的安装已使用（1）硬件篇

发表于 2016-06-25   |   分类于 Reference ， 其他   |     |   阅读次数

在Tesla计算卡的安装过程中，我学会了攒电脑……

材料

NVIDIA Tesla k40c计算卡一块
带有至少一个PCI-ex 16x的主板
800w以上电源一个
CPU，内存，硬盘 取高配

是否需要另一块独立显卡来做切换尚未确定，因为不知道能否实现交火
另：交火的过程需要额外学习

实验室硬件环境介绍

  • 电脑型号 华硕 All Series 台式电脑

  • 操作系统 Windows 8.1 Enterprise 64位 ( DirectX 11 )

  • 处理器 英特尔 Core i7-4790K @ 4.00GHz 四核

  • 主板 华硕 B85M-G ( 英特尔 Haswell )
  • 内存 32 GB ( 金士顿 DDR3 1866MHz )
  • 主硬盘 西数 WDC WD20EZRX-00D8PB0 ( 2 TB / 7200 转/分 )
  • 显卡 自带集成
  • 声卡 瑞昱 ALC887 高保真音频
  • 网卡 瑞昱 RTL8168/8111/8112 Gigabit Ethernet Controller / 华硕

主板一般，并且只有一PCI-e 16x ，以后扩展可能会很麻烦

除了接口，其他没啥用。

安装

和普通组装电脑没有大区别，只不过把pci-e本来的独立显卡换成加速卡，遗憾的是这样做的后果是显示屏失效（Tesla没有输出）

  1. 安装新电源
    关于电源，走访民间装机市场，他们给的建议是800w以上的，考虑到之后可能扩展，1000W 也是够用的，最终买的1300W，京东1000块。
  2. 连接电源线（注意：电源的模组化程度非常高，所以有哪个接头出现难插拔的情况，怎么回事儿？一定是接口方向什么的错了，呵呵），主要是主板，cpu，pci，硬盘.
  3. 安装加速卡Tesla
    插入PCI-e 16x插槽中，将6pin和8pin的电源插口接到电源提供的PCI接头.

原来之前担心的电源问题：怎么匹配设备的功率？这个担心完全用不上，因为电源的功率是大一些或者很多些是没有问题的，功率不足就不行了，根本带不动设备。

还有一点担心是：没有输出怎么安装驱动？这点在实验室的工作环境中得到了解决，原因是LAN中的其他计算机可以使用Windows的远程登录功能，操作该机器安装驱动和测试用例，这将在下一篇笔记中写。

至此，tesla加速卡在普通PC机硬件环境中的安装结束了，过程忘记拍照来了，有时间把他拆了重新装一遍。
